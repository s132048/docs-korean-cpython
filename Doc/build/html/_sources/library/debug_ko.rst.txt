**********************************************
디버깅과 프로파일링
**********************************************

이 라이브러리들은 파이썬 개발을 돕기 위한 것들이다.
디버거는 코드를 단계적으로 실행하고 스택 프레임을 분석하며 브레이크 포인트를 걸 수 있게 하는 등의 기능을 한다.
프로파일러는 코드를 실행시키고 실행 시간을 자세하게 쪼개어 프로그램 중의 병목을 발견할 수 있게 한다.
